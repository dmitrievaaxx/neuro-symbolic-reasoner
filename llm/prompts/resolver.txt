Этот файл содержит инструкции для модуля резолюций. Модуль 2 (движок резолюций) реализован на Python и не использует LLM - это чисто символьный алгоритм.

Инструкции для разработчика:
- Модуль принимает список формул в формате логики предикатов
- Выполняет алгоритм резолюций для поиска противоречия
- Возвращает лог шагов доказательства в формальном виде

Формат входных данных: список строк с формулами
Формат выходных данных: список шагов, каждый шаг содержит:
  - Номер шага
  - Описание операции (унификация, резолюция)
  - Результат операции
  - Финальный шаг: противоречие или отсутствие противоречия
