ПРАВИЛА ПРЕОБРАЗОВАНИЯ:
1. ОБРАБОТКА КВАНТОРОВ:
- Универсальные (∀): оставляй переменные
- Экзистенциальные (∃): вводи скулемовские константы (a, b, c...)
- Все переменные в итоговых клаузах считаются универсально квантифицированными

2. ТИПЫ ВЫСКАЗЫВАНИЙ:
- Конкретные факты:
"Сократ — человек" → "Человек(Сократ)"
- Общие утверждения:
"Все люди смертны" → "¬Человек(x) ∨ Смертен(x)"
- Экзистенциальные:
"Некоторые люди мудры" → "Человек(a)", "Мудр(a)" (две клаузы)
- Отрицательные:
"Ни один X не Y" → "¬X(z) ∨ ¬Y(z)"

3. РАСКРЫТИЕ ОПЕРАТОРОВ:
- Импликация:
A → B = ¬A ∨ B
- Конъюнкция:
A ∧ B ∧ C → ["A", "B", "C"] (отдельные клаузы)
- Дизъюнкция:
A ∨ B ∨ C → "A ∨ B ∨ C" (одна клауза)

4. ДОКАЗЫВАЕМОЕ УТВЕРЖДЕНИЕ:
- Берется ПОЛНОЕ ОТРИЦАНИЕ
- РЕКУРСИВНО ПРИМЕНЯЙ законы де Моргана пока не исчезнут все сложные отрицания
- УБЕРИ ВСЕ двойные отрицания: ¬(¬P) ⇒ P

ЗАКОНЫ ДЕ МОРГАНА:
- ¬(A ∧ B) = ¬A ∨ ¬B
- ¬(A ∨ B) = ¬A ∧ ¬B → преобразуется в ДВЕ клаузы: ["¬A", "¬B"]
- ¬(¬A) = A ← ОБЯЗАТЕЛЬНО ПРИМЕНЯЙ!

5. ФИНАЛЬНЫЕ ТРЕБОВАНИЯ:
- ТОЛЬКО атомарные предикаты с ¬ (без двойных отрицаний!)
- ТОЛЬКО ∨ между литералами
- НЕТ: ∧, →, ∀, ∃, скобок с отрицаниями
- Каждая клауза — отдельная строка
